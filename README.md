# 🚨 Women Safety App

A real-time emergency alert system built using React Native + Node.js + Twilio.

## 🔥 Features
- SOS Button
- Sends live
- SMS alert using Twilio
- Emergency call and sms trigger

## 🛠 Tech Stack
- React Native (Expo)
- Node.js + Express
- Twilio API

## ⚙️ Setup

### Backend
cd backend
npm install
node server.js

### Mobile App
cd mobile-app
npx expo start

## 📱 How it works
1. User presses SOS button
2. Location is captured
3. Backend sends SMS + Call via Twilio

## ⚠️ Note
This is a demo project (location may be simulated).
